# E-Commerce_Prototype
¿Qué estructura básica se requiere para el proyecto integrador?

mi-ecommerce-repo/
│
├── frontend/               <-- Aquí va tu proyecto de React
│   ├── public/
│   ├── src/
│   ├── package.json
│   └── ...
│
├── backend/                <-- (Opcional si usan Node/Python/PHP)
│   ├── server.js
│   └── ...
│
├── database/               <-- Aquí guardan sus scripts SQL
│   ├── script_creacion_tablas.sql
│   ├── datos_semilla.sql
│   └── diagrama_er.png
│
├── docs/                   <-- Documentación del equipo
│   ├── wireframes/
│   └── requerimientos.md
│
├── .gitignore              <-- ¡CRUCIAL! Para ignorar node_modules
├── README.md               <-- Instrucciones de instalación para el equipo
└── LICENSE
